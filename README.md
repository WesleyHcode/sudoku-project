# sudoku-project
Sudoku Project
by Wesley Ho

This program generates a solved Sudoku puzzle

Running the program:
1. Create Codespace on main
2. In "Terminal", enter javac SudokuGenerator.java SudokuGenTest.java
3. Then enter java SudokuGenTest

The puzzle is generated by getting a list of random integers from 1-9, placing them within the grid, checking to see if they are valid, then entering them into the board. If the last spot on the board is 0, it means that there must be a mistake in the number chosen, and so it reiterates back to use a different number.
